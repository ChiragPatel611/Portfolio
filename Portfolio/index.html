VoltWay

The Electric Vehicle Charging Station Finder is a web application designed to help EV users locate, book, and review nearby charging stations in real-time. The platform allows station hosts to register their charging points, manage availability, and set pricing. It features a responsive and user-friendly interface with role-based authentication for EV owners and hosts. The backend, powered by Spring Boot, is integrated with MongoDB to handle geospatial queries and manage user data securely. The application includes functionalities like real-time slot booking, geolocation-based station recommendations, and a review system to ensure service quality. Built using Angular for the frontend, it ensures seamless navigation across devices.
